## Short Description

Close empty Solana token accounts and reclaim rent in a clean mobile wallet flow.

## Full Description

Rent Closer is a simple Solana utility for reclaiming rent from empty SPL Token and Token-2022 accounts.

Connect a Solana Mobile compatible wallet, scan for empty token accounts, select the accounts you want to close, and approve the close-account transactions from your wallet. Reclaimed rent is returned to the connected wallet.

Rent Closer is non-custodial. It never asks for seed phrases or private keys. Transactions are prepared in the app and signed by the user's wallet.
